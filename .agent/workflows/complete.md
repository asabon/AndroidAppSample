---

## PR 作成フェーズ
1. ユーザーのプッシュ完了を確認
2. `gh pr create` を実行（タイトル、説明、ラベルを適切に設定）
3. ユーザーに PR の URL を提示し、マージを依頼

## クリーンアップフェーズ
マージ完了が報告されたら、以下の手順をナビゲートし、承認を得て実行します。

```bash
# 1. メインブランチへ移動
git checkout main

# 2. 最新のリモート変更を取得
git pull origin main

# 3. 作業が完了したローカルブランチを削除
git branch -d <branch-name>

# 4. リモートから削除されたブランチ情報を整理
git remote prune origin
```
